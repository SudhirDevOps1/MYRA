## 🎯 Pro Tips

1. **Use keyboard shortcuts** for power user workflow:
   - `Ctrl+N` = new chat
   - `Ctrl+F` = search messages
   - `Ctrl+L` = toggle mic

2. **Pin favorite cities** in Weather Dashboard for quick access

3. **Add prime contacts** in Settings for "close friend ko call karo"

4. **Set custom system prompt** to get consistent personality

5. **Switch providers** based on task:
   - Gemini for fast/free chat
   - Claude for long reasoning
   - GPT-4o for code
   - Grok for humor

6. **Enable streaming** for the best UX with long responses

7. **Use the Read Aloud (🔊)** button to listen to long AI responses while doing other work
